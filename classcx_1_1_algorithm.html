Detailed Description

More convenient Algorithm.

This is an extension of cv::Algorithm to provide simple ways to configure various parameters if its readParam() function is defined. The pre-built functions, setParam(), setParamValue(), and setParamText(), enable to set a value (scalar, vector, text) for each parameter through its name. Please refer its example at example.cpp.
